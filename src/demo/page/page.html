Sizes

1.用于设置按钮的尺寸大小，有两种内置的 size 尺寸，从小到大分别是：sm, default。

2.如果没有显示指定，将使用默认的 default 尺寸。

3.如果两种内置尺寸不符合你的需求，您还可以自定义尺寸，如上例，设置 theme="xxl"，会给按钮添加一个名为 nb-page-theme-xxl 的尺寸样式类。

4.页数较多时，如果想设置当前页码前后显示预备页码的个数，可以设置showCount(设置单边的个数即可)。第四个例子设置showCount为3
