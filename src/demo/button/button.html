nb-button

type: Directive

selector: button[nb-button], a[nb-button]

exportAs: nbButton

Usage

import {ButtonModule} from 'xdesign';

Examples

Themes

1. Button theme, there are four built-in themes: primary, default, neutral, transparent. If the theme property is not specified, the default theme will take effect.

2. If the disabled property is set, the disabled theme will take effect first.

3. You can also customize your own theme. e.g. theme="yellow"ï¼Œwill add a wrapper class nb-button-theme-yellow on button.

Sizes

1. Button size, there are four built-in sizes(from small to large): xs, sm, default, lg. If not specified, the default size will take effect.

2. You can also customize your own size. e.g. size="xxl", will add a wrapper class nb-button-size-xxl on button.

Link Button

nb-button can be used on <a> links.

Button with icons

1. Because the button content utilises ng-content, it's convenient to add icons.

2. You can use built-in icons or third-party icons, e.g. fontawesome.

3. Combined with the theme and size property, round icon button is easy to define.
