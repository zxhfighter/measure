nb-breadcrumb

type: Component

selector: nb-breadcrumb

exportAs: nbBreadcrumb

Usage

import {BreadcrumbModule} from 'xdesign';

Examples

Breadcrumb Examples

nb-breadcrumb used to show the current site level, and navigate quickly. Two modes are supported, one is href navigation(set the href property in datasource), the other is router navigation(set the path property in datasource).
