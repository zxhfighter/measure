Usage

import {GridModule} from 'xdesign';

Examples

基础布局

从堆叠到水平排列。
使用单一的一组 nb-row 和 nb-col 栅格组件，就可以创建一个基本的栅格系统， 所有列（nb-col）必须放在 nb-row 内。

区块间隔

栅格常常需要和间隔进行配合，你可以使用 nb-row 的 nbGutter 属性，我们推荐使用 (16+8n)px 作为栅格间隔。

左右偏移

列偏移
使用 nbOffset 可以将列向右侧偏。例如，[nbOffset]="'4'" 将元素向右侧偏移了 4 个列（column）的宽度。

布局排序

列排序
使用 nbPull 和 nbPush 类就可以很容易的改变列（column）的顺序。

Flex布局

Flex布局
使用 [nbType]="'flex'" 定义 flex 布局，其子元素根据 [nbJustify] 不同的值 start,center,end,space-between,space-around，分别定义其在父节点里面的排版方式。

Flex对齐

Flex对齐
Flex 子元素垂直对齐。

响应式布局

参照 Bootstrap 的 响应式设计 ，预设四个响应尺寸：xs sm md lg xl
